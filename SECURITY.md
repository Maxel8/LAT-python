# Security Policy

## Supported Versions


| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark:|

## Reporting a Vulnerability

If you discover a security vulnerability, please do NOT open a public issue.

Instead, use the following formula: [Security Advisories](https://github.com/Maxel8/LAT-python/security/advisories/new "GitHub Security Advisories - LAT-python")


Please include:
- Description of the issue
- Steps to reproduce
- Possible impact
